====== CreateWebsite ======

===== Manually =====
to create the website from within zim do
* File -> Export...

settings
* Complete Notebook
* HTML, default

export to
* ./doc/web

index page
* -leave blank-

===== Makefile =====
there is a makefile to do this at ./doc/Makefile
